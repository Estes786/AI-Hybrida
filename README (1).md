## 🚀 Quick Start

### 1. **Clone Repository**
```bash
git clone https://github.com/estes786/quantum-ai-hybrida.git
cd quantum-ai-hybrida
```

### 2. **Local Development**
```bash
# Simple HTTP server
python -m http.server 8000
# or
npx serve .
```

### 3. **Deploy to Vercel**
```bash
npm install -g vercel
vercel --prod
```
